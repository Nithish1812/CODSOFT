## 🚀 Projects Overview

### Task 1: To-Do List Application
[cite_start]A command-line based application that helps users manage and organize their tasks efficiently[cite: 41, 42].

#### Features:
* **Add Tasks:** Create new entries in your list.
* **View Tasks:** Display all current tasks with their index numbers.
* **Delete Tasks:** Remove completed or unwanted tasks by their number.
* **Persistent Storage:** All tasks are automatically saved to and loaded from a local `tasks.txt` file.
